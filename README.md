# BitBenders 🚀

> Built for the hackathon. Ready to ship.

## Getting Started

### Requirements
- [Node.js](https://nodejs.org/) (v18 or higher)
- [Git](https://git-scm.com/)

### Setup (Mac & Windows)

```bash
# 1. Clone the repo
git clone https://github.com/thevinayak26/BitBenders.git

# 2. Go into the project folder
cd BitBenders

# 3. Install dependencies (once a project is added)
npm install

# 4. Start the app
npm run dev
```
